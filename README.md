# RaceStream

RaceStream es una aplicación web orientada a la centralización de información relacionada con competiciones de automovilismo.

## Tecnologías
- Java
- Spring Boot
- MySQL
- JPA/Hibernate
- HTML, CSS y JavaScript
- OpenF1 API

## Estado actual
- CRUD de campeonatos y circuitos
- Relación entre carreras, campeonatos y circuitos
- Demo funcional con OpenF1
- Páginas principales: admin.html, races.html y f1.html

## Próximas fases
- Usuarios
- Autenticación
- Roles
- Favoritos
- Notificaciones

## Ejecución
1. Crear una base de datos MySQL llamada `racestream_db`
2. Configurar `src/main/resources/application.properties`
3. Ejecutar el proyecto con Maven o desde el IDE

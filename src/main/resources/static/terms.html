Términos y condiciones

Condiciones de uso de RaceStream como plataforma académica de consulta y visualización de Fórmula 1.

Uso permitidoLa web se puede usar para consultar datos públicos de la temporada actual, iniciar sesión, guardar favoritos, participar en el foro y enviar reportes desde una cuenta registrada.
Datos deportivosLos datos proceden de APIs externas y pueden sufrir retrasos o correcciones. RaceStream no sustituye comunicados oficiales.
Fuentes oficialesRaceStream consulta datos de OpenF1, Jolpica F1, F1DB y GNews. Cada fuente conserva sus propias condiciones, disponibilidad y posibles cambios de datos.
ModeraciónEl administrador puede moderar publicaciones del foro y bloquear usos abusivos para proteger la plataforma.
Normas del foroEl foro debe usarse con respeto. No se permite spam, contenido ofensivo, ilegal, discriminatorio o peligroso, ni publicar datos personales de terceros. RaceStream puede moderar o eliminar contenido y las publicaciones pueden quedar asociadas a la cuenta del usuario.
Normas de contactoEl formulario de contacto se usa para soporte, comunicación sobre incidencias y mejoras de RaceStream. No debe usarse para spam ni para enviar datos sensibles innecesarios.
DisponibilidadLa aplicación puede tardar más si una API externa responde lento. RaceStream prioriza no mostrar resultados incompletos.
Acceso sin cuentaLos visitantes pueden consultar contenido básico de la temporada actual. En Vivo, foro, contacto, favoritos, preferencias, filtros históricos y gestión de cuenta requieren iniciar sesión.
Favoritos y avisosLos favoritos sirven para acceder rápido a pilotos, escuderías y GP. Si el usuario activa preferencias de app o correo, RaceStream puede avisar del inicio de sesiones favoritas y mostrar un resumen cuando finalicen.
Responsabilidad del usuarioCada usuario debe proteger sus credenciales y cerrar sesión en equipos compartidos.

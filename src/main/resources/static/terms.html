Términos y condiciones

Condiciones de uso de RaceStream como plataforma académica de consulta y visualización de Fórmula 1.

Uso permitidoLa web se puede usar para consultar datos, guardar favoritos, participar en el foro y enviar reportes desde una cuenta registrada.
ConductaNo se permite publicar contenido ofensivo, suplantar identidades, hacer spam o intentar alterar servicios, APIs o datos.
Datos deportivosLos datos proceden de APIs externas y pueden sufrir retrasos o correcciones. RaceStream no sustituye comunicados oficiales.
ModeraciónEl administrador puede moderar publicaciones del foro y bloquear usos abusivos para proteger la plataforma.
Proyecto académicoRaceStream forma parte de un TFG y puede evolucionar con nuevas funciones, ajustes técnicos y mejoras de seguridad.
